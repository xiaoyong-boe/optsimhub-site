# 随机纹理生成

> 生成随机无序点阵，适用于光学仿真与纹理设计

## 什么是随机纹理生成？

一款在线工具，能够基于晶格算法生成各类随机分布的无序点阵，满足科研与设计需求。

## 核心功能

- **晶格算法** — 基于先进晶格算法，生成高质量随机无序点阵
- **参数可调**
  - 点数量
  - 区域大小
  - 最小间距
  - 分布参数

- **导出格式**
  - DXF 文件（CAD 兼容）
