# 随机纹理生成

> 生成随机无序点阵，适用于光学仿真与纹理设计

## 什么是随机纹理生成？

一款在线工具，能够生成各类随机分布的无序点阵，满足科研与设计需求。

## 核心功能

- **多种分布类型** 
  - 均匀随机分布
  - 高斯分布
  - 泊松分布
  - 蓝噪声分布
  
- **参数可调**
  - 点数量
  - 区域大小
  - 最小间距
  - 分布参数

- **导出格式**
  - PNG 图片
  - CSV 数据
  - JSON 数据

## 适用场景

- 光学散斑仿真
- 随机介质建模
- Monte Carlo 模拟
- 纹理图案设计
- 粒子系统背景
