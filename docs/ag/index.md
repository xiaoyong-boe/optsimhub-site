# AG 生成

> AI 驱动的光学仿真自动生成平台

## 什么是 AG 生成？

AG 生成是一款基于人工智能的自动化建模工具，
能够根据用户需求自动生成定制化的光学仿真方案。

## 核心功能

- **AI 自动建模** — 描述你的需求，AI 自动设计膜层结构
- **参数优化** — 智能调参，找到最优解
- **多目标优化** — 同时满足多个性能指标
- **快速迭代** — 分钟级生成设计方案
- **一键导出** — 支持主流仿真软件格式

## 适用场景

- 薄膜干涉设计
- 偏振器件优化
- 光学涂层设计
- 光谱响应调控
