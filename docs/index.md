## 最新软件

### 邦加球 V1

偏振光学仿真工具

**容差分析** — 基于 TMM + 严格耦合波方法，对膜层厚度和折射率注入正态分布扰动，运行大规模 Monte Carlo 仿真。单波长模式输出均值、标准差、P5/P95 分位区间及工艺良率；波长扫描模式生成名义曲线 + 置信包络，直观显示敏感波段。支持独立种子复现，方便 A/B 对比不同容差方案，助力设计与制造的最优平衡。
- **Poincaré 球可视化** — 实时光的偏振态在球面上的轨迹追踪
- **Mueller 矩阵计算** — 完整 Jones → Mueller 矩阵转换
- **Berreman 4×4 求解器** — 各向异性多层膜精确计算
- **TMM transfer-matrix** — 薄膜光学正演仿真
- **PyQt6 GUI** — 交互式参数调节，实时绘图
