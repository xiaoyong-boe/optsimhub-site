# polarization_poincare

偏振光学仿真工具，基于 transfer-matrix method (TMM) 和 Berreman 4×4 求解器。

## 功能特性

- **Poincaré 球可视化** — 实时光的偏振态在球面上的轨迹追踪
- **Mueller 矩阵计算** — 完整 Jones → Mueller 矩阵转换
- **Berreman 4×4 求解器** — 各向异性多层膜精确计算
- **TMM transfer-matrix** — 薄膜光学正演仿真
- **PyQt6 GUI** — 交互式参数调节，实时绘图
