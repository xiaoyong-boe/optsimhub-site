# 使用说明

## 快速开始

1. 启动软件后，进入 **参数设置** 面板
2. 设置入射角、波长、膜层结构
3. 点击 **计算** 查看 Poincaré 球轨迹和 Mueller 矩阵结果

## 膜层结构配置

```
[入射介质] → [膜层1] → [膜层2] → ... → [基底]
```

每层膜需要设置：
- **厚度** (nm)
- **折射率** n (复数输入支持吸收介质)
- **各向异性** (optional)

## 偏振光源配置

支持以下偏振态定义：
- **线偏振** — 设置方位角 ψ
- **圆偏振** — 设置椭圆率角 χ
- **椭圆偏振** — 同时设置 ψ 和 χ

## 导出数据

计算结果支持导出为：
- PNG 图片 (Poincaré 球截图)
- CSV (Mueller 矩阵数值)
